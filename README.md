# VOD Platform Database Schema

## Overview
This repository contains a relational database schema for a fictional VOD (Video on Demand) streaming platform. It was originally created as an academic group project and includes randomly generated dummy data for testing purposes.

## What's Inside
* **SQL Architecture:** Fundamental database components including table relations, triggers, views, stored procedures, and scheduled events used to manage users, subscriptions, and content viewing history.
* **Recommendation Script:** A basic Python script that connects to the MySQL database to recommend movies based on a user's most frequently watched genre.

## Disclaimer
This project is strictly for portfolio and educational purposes. All user names, emails, and activities are synthetic and randomly generated.
